📖 VNLog 임베디드 컴포넌트

HTML 페이지에 간편하게 삽입할 수 있는 비주얼 노벨 플레이어

🎯 소개

VNLog 임베디드 컴포넌트는 TRPG 로그를 비주얼 노벨 형식으로 재생하는 플레이어 전용 웹 컴포넌트입니다. 외부 CSS 영향을 받지 않으며, 어떤 웹페이지에도 삽입 가능합니다. (편집은 VNLog 앱의 편집기에서 하고, 내보낸 JSON을 임베드하세요.)

  • 플레이어 전용: 파싱된 JSON 데이터를 재생
  • 스타일 격리: scoped 스타일로 외부 CSS 간섭 방지
  • 반응형 디자인: 모바일/태블릿/데스크탑 모두 지원

▶ 플레이어 모드 사용법

이미 파싱된 JSON 데이터를 재생만 하는 경우 플레이어 컴포넌트를 사용하세요.

<!-- HTML에 삽입 --> <div id="vnlog-player"></div> <script> // Vue 앱 초기화 const app = Vue.createApp({ template: '<VNLogEmbedPlayer :vn-data="vnData" />', data() { return { vnData: { title: "TRPG 로그 제목", scenes: [ { id: "scene1", name: "씬 1", steps: [ { id: "step1", character: { name: "캐릭터A", color: "#ff6b6b", avatarUrl: "" }, text: "안녕하세요!", effects: {} } ] } ], characters: {}, handouts: [] } } } }) app.component('VNLogEmbedPlayer', VNLogEmbedPlayer) app.mount('#vnlog-player') </script>
⚠️ 주의사항 플레이어 모드는 vnData props로 JSON 데이터를 전달받아야 합니다.
데모 (플레이어)

🚀 빌드 및 배포

프로젝트를 빌드하여 단일 JS 파일로 만들 수 있습니다.

# Vite 빌드 (라이브러리 모드) npm run build:embed # 생성된 파일 dist/vnlog-embed.js dist/vnlog-embed.css

빌드된 파일을 외부 웹사이트에 포함:

<link rel="stylesheet" href="https://your-cdn.com/vnlog-embed.css"> <script src="https://cdn.jsdelivr.net/npm/vue@3"></script> <script src="https://your-cdn.com/vnlog-embed.js"></script>
